# 📊 交互式数据可视化应用

基于Jupyter Notebook数据的现代化Web可视化应用，提供丰富的交互功能和响应式设计。

## 🚀 项目概述

本项目将Jupyter Notebook中的静态数据可视化图表转换为功能完整的交互式Web应用，包含6个不同类型的数据可视化实例，支持多种交互操作和数据导出功能。

## ✨ 核心功能

### 📈 数据可视化
- **6个完整实例图表**：电影票房、支付宝消费、英语成绩、速度距离、阿里巴巴GMV、面包配料
- **多种图表类型**：条形图、饼图、散点图、分组条形图
- **响应式设计**：适配桌面、平板、手机等不同设备

### 🎮 交互功能
- **图表筛选**：按类型筛选显示特定图表
- **数据过滤**：关键词搜索筛选数据
- **缩放浏览**：每个图表独立缩放控制
- **悬停提示**：鼠标悬停显示详细数据信息

### 💾 数据导出
- **PNG格式**：导出高清图表图片
- **CSV格式**：导出原始数据表格
- **社交媒体分享**：Twitter、Facebook、LinkedIn分享

### 🎨 用户体验
- **CSS动画**：平滑的过渡效果和悬停动画
- **JavaScript动画**：图表加载动画和交互反馈
- **现代化UI**：简洁美观的界面设计

## 🛠️ 技术栈

- **前端框架**：React 17
- **图表库**：Chart.js
- **样式技术**：CSS3 + Flexbox + Grid
- **导出功能**：html2canvas
- **开发工具**：Babel (浏览器端编译)

## 📊 数据实例

### 1. 电影票房排行榜
- **类型**：水平条形图
- **数据**：2019年内地电影票房Top15
- **特点**：票房数据对比展示

### 2. 支付宝消费分析  
- **类型**：饼图
- **数据**：各类消费金额比例
- **特点**：消费结构可视化

### 3. 英语成绩对比
- **类型**：分组条形图
- **数据**：高二年级各班男女生英语平均成绩
- **特点**：性别成绩对比分析

### 4. 速度与制动距离
- **类型**：散点图
- **数据**：汽车速度与制动距离关系
- **特点**：物理关系可视化

### 5. 阿里巴巴GMV数据
- **类型**：条形图
- **数据**：2013-2019财年淘宝天猫GMV
- **特点**：时间序列增长趋势

### 6. 果酱面包配料比例
- **类型**：饼图
- **数据**：面包制作配料重量比例
- **特点**：配方比例可视化

## 🎯 使用指南

### 基本操作
1. **选择图表类型**：使用下拉菜单筛选特定图表
2. **数据筛选**：在搜索框输入关键词过滤数据
3. **图表缩放**：点击图表右上角的 +/- 按钮进行缩放
4. **查看详情**：鼠标悬停在图表元素上查看详细数据

### 数据导出
1. **导出PNG**：点击对应图表的"导出图表(PNG)"按钮
2. **导出CSV**：点击"导出数据(CSV)"按钮下载原始数据
3. **分享图表**：使用社交媒体按钮分享当前页面

### 响应式适配
- **桌面端**：多列网格布局，最佳浏览体验
- **平板端**：自适应单列布局，保持可读性
- **手机端**：优化触摸交互，简化界面元素

## 🔧 技术特点

### 性能优化
- **懒加载**：图表按需渲染，提升加载速度
- **代码拆分**：模块化组件设计，便于维护
- **内存管理**：图表实例正确销毁，避免内存泄漏

### 兼容性
- **浏览器支持**：Chrome、Firefox、Safari、Edge等现代浏览器
- **移动端适配**：完善的触摸事件处理和响应式布局
- **无障碍访问**：基本的键盘导航和屏幕阅读器支持
